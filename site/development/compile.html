Compile

Once all requirements are up and working, just type:

mvn clean package

If you want to skip the tests type:

mvn clean package -DskipTests=true

First compilation will take longer due to the process of downloading all dependencies. If there are no problems, you can install the artifact in your local maven repository:

mvn install -DskipTests=true

Documentation is bundled with the source code. Generating it from source code is as difficult as executing: 

mvn clean site:site -DskipTests=true

Documentation can be accessed from \target\site\index.html.
